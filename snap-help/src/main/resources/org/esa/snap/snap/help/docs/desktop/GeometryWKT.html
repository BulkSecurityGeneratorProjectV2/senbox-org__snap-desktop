Geometry from WKT

This command allows to copy a geometry from Well-Known-Text (WKT) format. It can be used as a parameter for the binning processor or in external tools and applications.

WKT from Geometry

This command allows to create a geometry from Well-Known-Text (WKT) format. It is only enabled if a geometry is selected.
