Tool Windows

This menu item provides a list of available tool windows you can activate.


Status Bar

Here you toggle the visibility of the status bar, which is located in the downmost area of the Sentinel Toolbox main window. The status bar is used to display tool tips, the RAM consumption of the Toolbox and a button where you are able to force a garbage collection (i.e. free unused memory). Deselecting the check box here will remove the status bar from the main window (and vice versa).


Synchronize Image Cursors

Synchronizes cursor positions across multiple image windows. The functionality is also available if you click on in the Navigation Window.

Synchronize Image Views

Synchronizes views across multiple image windows. The functionality is also available if you click on in the Navigation Window.
