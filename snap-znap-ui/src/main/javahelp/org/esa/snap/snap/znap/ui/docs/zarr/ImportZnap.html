Import ZNAP

This command allows to import the data of a ZNAP format.

ZNAP data is either written as a single zip archive with the extension ".znap.zip" or written in several files within a data folder with the extension ".znap".

To open a product select either the zip archive (with extension ".znap.zip") or any file inside the data folder with the extension ".znap" in the file chooser. Alternatively you can also Drag-and-Drop either the data folder or the zip archive onto the Product Explorer in SNAP.

For more details regarding the format and write options see: ZNAP writer.
