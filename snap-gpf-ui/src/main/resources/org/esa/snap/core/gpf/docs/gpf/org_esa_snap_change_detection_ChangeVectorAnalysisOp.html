ChangeVectorAnalysisOp Operator

ChangeVectorAnalysisOp Operator Description

Overview

Name: ChangeVectorAnalysisOp
Full name: org.esa.snap.change.detection.ChangeVectorAnalysisOp
Purpose: The 'Change Vector Analysis' between two dual bands at two differents dates.
Version: 1.0

Description

The ChangeVectorAnalysisOp is change vector analysis operation between two dual bands at two differents dates. Return two products magnitude and direction of change

Sources

Name Description
sourceProduct1 The sources product at the first date.
sourceProduct2 The sources product at the second date.

Parameters

Name Data Type Default Description Constraints
sourceBand1 String Band 1 at the same date
sourceBand2 String Band 2 at the same date
magnitudeThreshold String 0 No change detection magnitude threshold
